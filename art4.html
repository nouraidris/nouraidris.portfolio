STATEMENT CONTACT PHOTOGRAPHY PORTFOLIO


Labeled (2020)

Glitch

Description

I used this photo that I took of my brother and glitched it using a digital audio editor. First, I converted the photo into black and white and changed the file format to tiff in order to be able to import it into audacity. Then, I manipulated the data of the image using different remix settings which created the glitch effect. Lastly, I reverted the notes back to a JPEG format and came up with this final image. I named this piece “labeled” because in the photo my brother is seen smoking. People are conditioned to label others based on actions or appearances without having a real and clear image of the person.
